冒泡排序 (Bubble Sort)

比较相邻元素，如果顺序错误就交换它们。这一过程会重复 直到不需要交换为止。

速度:
准备就绪
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        # 标记是否发生交换
        swapped = False
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:
            break
