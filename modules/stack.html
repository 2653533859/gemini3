栈 (Stack)

后进先出 (LIFO - Last In First Out) 的线性数据结构。 想象一摞盘子，只能从顶部添加或取下。

Max Size: 8
← TOP
Stack is empty
class Stack:
    def __init__(self):
        self.items = []
    def push(self, item):
        self.items.append(item)
    def pop(self):
        if not self.items: return None
        return self.items.pop()
    def peek(self):
        if not self.items: return None
        return self.items[-1]
