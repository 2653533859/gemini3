插入排序 (Insertion Sort)

构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。

速度:
准备就绪
def insertion_sort(arr):
    n = len(arr)
    # 从第1个元素开始（默认第0个已有序）
    for i in range(1, n):
        key = arr[i]
        j = i - 1
 
        # 将比 key 大的元素向后移动
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
 
        # 插入 key
        arr[j + 1] = key
