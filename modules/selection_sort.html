选择排序 (Selection Sort)

每次从未排序区间中找到最小的元素，将其放到已排序区间的末尾。

速度:
准备就绪
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        # 假设当前位置 i 为最小值索引
        min_idx = i
        for j in range(i+1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
 
        # 将找到的最小值交换到位置 i
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
