深度优先搜索 (DFS) - 网格寻路

使用栈 (Stack) 策略。DFS 会一条路径走到黑，直到不能再走为止才回溯。 注意：DFS 找到的通常不是最短路径。

起点: 绿色 终点: 红色

def dfs(grid, curr, target):
    if curr == target: return True
    if curr in visited: return False
    visited.add(curr)
    for neighbor in get_neighbors(curr):
        if dfs(grid, neighbor, target):
            return True
    return False
