二叉搜索树 (BST)

左子树所有节点 < 根节点 < 右子树所有节点。
支持 O(log n) 的快速查找、插入和删除。

class TreeNode:
    def __init__(self, val):
        self.val = val; self.left = None; self.right = None
def insert(root, val):
    if not root: return TreeNode(val)
    if val < root.val:
        root.left = insert(root.left, val)
    else:
        root.right = insert(root.right, val)
    return root
