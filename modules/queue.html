队列 (Queue)

先进先出 (FIFO - First In First Out) 的线性数据结构。 像排队买票一样，先来的先服务。

HEAD (队头)
TAIL (队尾)
Queue is empty
class Queue:
    def __init__(self):
        self.items = []
    def enqueue(self, item):
        self.items.append(item)
    def dequeue(self):
        if not self.items: return None
        return self.items.pop(0)
    def peek(self):
        if not self.items: return None
        return self.items[0]
