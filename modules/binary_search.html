二分查找 (Binary Search)

在 **有序数组** 中查找特定元素的算法。每次比较中间元素，将搜索范围缩小一半。 时间复杂度: O(log n)。

准备就绪
def binary_search(arr, target):
    low = 0
    high = len(arr) - 1
 
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid # Found
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
 
    return -1 # Not Found
