实战：爬楼梯 (Climbing Stairs)

LeetCode #70. 假设你正在爬楼梯。需要 n 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。有多少种不同的方法可以爬到楼顶？
核心思路：动态规划 (DP)。到达第 i 阶的方法数 = 到达第 i-1 阶的方法数 + 到达第 i-2 阶的方法数。

dp[i] = dp[i-1] + dp[i-2]
DP Table (记忆化数组)
准备就绪
def climbStairs(n):
    if n <= 2: return n
    dp = [0] * (n + 1)
    dp[1] = 1; dp[2] = 2
    for i in range(3, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]
