实战：有效的括号 (Valid Parentheses)

LeetCode #20. 给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。
核心思路：遇到左括号入栈，遇到右括号检查栈顶是否匹配。

准备就绪
def isValid(s):
    stack = []
    pairs = {")": "(", "]": "[", "}": "{"}
    for char in s:
        if char in pairs:
            if not stack or stack[-1] != pairs[char]:
                return False
            stack.pop()
        else: stack.append(char)
    return not stack
