模块一：基础排序

冒泡排序 (Bubble Sort)
通过重复交换相邻的逆序元素，像气泡一样将最大值“浮”到顶端。
包含 Python 代码对照
选择排序 (Selection Sort)
每次从未排序部分选择最小值放到已排序部分的末尾。
包含 Python 代码对照
插入排序 (Insertion Sort)
构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。
包含 Python 代码对照
快速排序 (Quick Sort)
分治法经典应用。包含 Partition 过程可视化。
包含 Python 代码对照
归并排序 (Merge Sort)
稳定排序。递归地将数组分成两半，再合并有序数组。
包含 Python 代码对照
